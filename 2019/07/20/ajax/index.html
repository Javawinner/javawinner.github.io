报错信息

报错1

Uncaught TypeError

原因： 前端部分 在发送ajax请求时候未取到表单输入的值,找到标签后没有加 .val取值

报错2

Uncaught ReferenceError

原因：ajax没有接受后台传送的值

报错3

Failed to load resource: the server responded with a status of 500 (Internal Server Error)

原因: 视图函数未返回http协议的数据
